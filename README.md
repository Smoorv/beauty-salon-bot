# Beauty Salon Telegram Bot
Бот для записи в салон красоты с функционалом:
- Прайс-лист
- Контакты
- Заявки на посещение
  
Технологии:
- Telebot
- Sqlite3

## Запуск
1. Установите зависимости: `pip install -r requirements.txt`
2. Создайте `.env` с токеном бота: `TOKEN=ваш_токен`
3. Запустите: `python bot.py`
